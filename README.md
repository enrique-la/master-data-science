# Data Science — Máster en IA & Data Science

Asignatura de fundamentos estadísticos y programación para Data Science.

## Contenido

### Estadística & R
- Estadística descriptiva e inferencial
- Tests de hipótesis, bootstrap
- Regresión lineal y polinomial
- Introducción a R y RMarkdown

### Python para Data Science
- NumPy, Pandas, Matplotlib, Seaborn
- Análisis exploratorio de datos (EDA)
- Visualización de datos

### Introducción al Machine Learning
- Métricas de evaluación: MSE, Accuracy, MCC, ROC, Precision-Recall
- Bias-Variance tradeoff
- Validación cruzada y regularización
